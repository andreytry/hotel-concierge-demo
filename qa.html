Fires every test case at the live /webhook/concierge endpoint (real GPT-4o agent, real tenant facts), captures the actual reply, and scores it against the rubric. Runs in your browser; nothing is faked.

Test schema & scoring rubric

Dimensions scored per turn: (1) Relevance/coverage — did it actually answer using real knowledge? (2) Language fidelity — did it reply in the language asked? (3) Grounding — answer consistent with hotel facts, no invented amenities. (4) Booking behavior — on booking intent, does it move toward a direct booking (asks dates/guests, quotes rate, mentions direct perk)?

Verdicts: PASS answered correctly & in the right language · PARTIAL partial / generic / right idea but thin · FAIL deflected when it should answer, wrong language, error, or fabricated.

Categories: neighborhood (55 EN), hotel_fact (30 EN), booking (15 EN), multilang (25: 9 ES / 8 DE / 8 FR). Heuristic verdict is a guide — the raw reply is shown for every case so you can audit it yourself.
